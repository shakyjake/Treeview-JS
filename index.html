Treeview JS

Creates a tree view from an unordered list

Example:

  • Apples
    • Granny Smith
    • Golden Delicious
    • Pink Lady
    • Pine
  • Grapes
    • Seedless
      • Green
      • Red
    • Seedful
      • Green
      • Red
  • Pears
  • Coconut
    • Lime

Usage:

Include the Treeview JS script somewhere in the page and create the treeview with "new TreeView(Selector);". Selector must match the first UL in the list.

<script src="treeview/treeview.0.2.js"></script> <script> (function(){ new TreeView('#TreeView'); })(); </script>

You can also create multiple trees in the same page (with the same selector) and have branches open by default by adding the "open" class to an LI
